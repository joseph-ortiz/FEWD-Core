FEWD-Core

I Created this project becauase I got tired of always setting up other dev tools such as gulp, jslint, sass , handlebars...

So instead of giving up on these tools, I'll just make one project that I can use as a template. This will make my dev work much easier :)

Modules included

  • npm
  • Gulp
  • Browser-Sync
  • Sass
  • uglify
  • rename
  • concat
